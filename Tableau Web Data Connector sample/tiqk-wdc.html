TIQK Web Data Connector for Tableau

This is a sample Web Data Connector. It is designed to showcase how to configure access to TIQK organisation and audit result data in Tableau.

Before running, configure the apiKey and apiSecret values in the tiqk-wdc.js file. Your apiKey and apiSecret can be generated and viewed by logging into your TIQK account at tiqk.io, under the Your organisation > Integrations section.

If running this on a local machine in using the Tableau Web Data Connector Simulator, you may need to install and activate the 'CORS Everywhere' extension for Firefox and Chrome to avoid triggering CORS errors.
